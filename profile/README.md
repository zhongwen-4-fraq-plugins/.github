## 插件

| 插件 | 功能 | 安装 |
| --- | --- | --- |
| [fraq-plugin-github](https://github.com/zhongwen-4-fraq-plugins/fraq-plugin-github) | 连接 GitHub App、GitHub API 与 QQ 群，支持事件订阅、仓库查询以及 Issue 和 Pull Request 操作 | `pnpm add fraq-plugin-github` |
| [fraq-plugin-group-admin](https://github.com/zhongwen-4-fraq-plugins/fraq-plugin-group-admin) | 提供入群审核、刷屏处理、黑白名单、群文件分类和群名片管理等群管能力 | `pnpm add fraq-plugin-group-admin` |

## 开始使用

每个插件的环境要求、配置方式和命令说明，请查看对应仓库的 README。

这些插件使用 TypeScript 开发，适用于基于 Fraq 构建的 QQ 机器人。具体的 Node.js 和 Fraq 版本要求以各插件文档为准。
